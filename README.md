# 基于 LabVIEW 的虚拟仪器系统  
Virtual Instrument System Based on LabVIEW

---

## 项目简介

本项目基于 **LabVIEW** 平台，实现了两个典型虚拟仪器系统：  

#### 本项目为大学生虚拟仪器课程实验，仅供学习和参考，祝大家学有所成！

- **虚拟信号发生器（Signal Generator）**
- **虚拟双通道示波器（Dual-channel Oscilloscope）**
---

---
## 功能说明
### 虚拟信号发生器
- 支持多种基础信号波形生成  
- 支持参数调节（幅值、频率、相位等）
- 实时波形显示（图 / 图表模式）
- 适合信号源与系统输入实验

### 虚拟双通道示波器
- 双通道信号同步采集与显示
- 支持基本时域观测
- 用于对比分析不同信号或输入/输出关系
